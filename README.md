# ESP32 Beginner Guide

A complete beginner-friendly guide to getting started with the ESP32 Development Board using the Arduino IDE.

This repository accompanies the tutorials published by **Chip.pk** and includes sample code, wiring diagrams, setup instructions, and project resources.

---

# Overview

The ESP32 is a powerful Wi-Fi and Bluetooth-enabled microcontroller widely used for IoT, robotics, automation, smart home, and embedded systems.

This guide will help you:

- Install the Arduino IDE
- Install the ESP32 Board Package
- Connect the ESP32 to your PC
- Upload your first program
- Understand the basic workflow
- Start building IoT projects

---

# Hardware Required

- ESP32 Development Board
- USB Data Cable

# Getting Started

1. Install Arduino IDE.
2. Install the ESP32 Board Package.
3. Connect the ESP32 using a USB cable.
4. Select the correct board and COM port.
5. Upload the Blink example.
6. Open the Serial Monitor.
